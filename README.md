# My Valentine

A modern, interactive Valentine's Day proposal website built with React, Tailwind CSS, and Framer Motion.

## Features

- **Interactive "Yes/No" Buttons**: The "No" button playfully runs away when hovered or clicked.
- **Growing "Yes" Button**: Encourages the user to say yes!
- **Animations**: Smooth transitions and floating hearts using Framer Motion.
- **Sound Effects**: Custom audio synthesis (Web Audio API) for interactions.
- **Confetti**: A celebration effect upon acceptance.
- **Responsive Design**: Works on desktop and mobile.

## Setup

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Structure

- `src/App.jsx`: Main application logic.
- `src/main.jsx`: Entry point.
- `src/index.css`: Tailwind imports and custom styles.
- `index.html`: HTML entry point.
- `index.backup.html`: Backup of the original website.
